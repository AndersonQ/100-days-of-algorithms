Division
========

A complete division algorithm using additions, subtractions, and comparisons

## Build
```
$ go build
```

## Run
```
$ ./015.Division.Algorithm -n [NUMERATOR] -d [DENOMINATOR]
$ ./015.Division.Algorithm -n 42 -d 21
```

## Test
```
$ go test -cover ./maths
```
