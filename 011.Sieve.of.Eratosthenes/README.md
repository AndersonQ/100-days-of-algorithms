Sieve Of Erastosthenes - Finding Primes
======================

Finding the `n` first primes using the [Sieve of Eratosthenes](https://en.m.wikipedia.org/wiki/Sieve_of_Eratosthenes) algorithm

## Build 
 
``` 
$ go build 
 
``` 
 
## Run 
 
 
``` 
$ ./011.Sieve.of.Eratosthenes N
  
```
