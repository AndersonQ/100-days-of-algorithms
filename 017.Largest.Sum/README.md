Largest Sum
===========

A wee program to find the largest sum in an int array

## Build
```
$ go build
```

## Run
```
$ ./013.Shuffle [ARRAY]
```

To find the largest sum in *2 -4 2 -1 3 -3 10 -1 -11 -100 8 -1*

```
$ ./017.Largest.Sum 2 -4 2 -1 3 -3 10 -1 -11 -100 8 -1
```

## Test
```
$ go go test -cover ./maths
```
